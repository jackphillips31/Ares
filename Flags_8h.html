Detailed Description

This file defines various window settings flags that are used to specify the behavior and appearance of the window when creating a new window for the application. The flags can be combined to set properties such as windowed mode, fullscreen mode, borderless, resizable, etc. The file also defines convenient constants for the most common window settings.
