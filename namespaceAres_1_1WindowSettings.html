Detailed Description

This namespace defines various flags that can be used to configure the behavior and appearance of the window. The flags are used to specify window properties such as windowed mode, fullscreen, and resizability.

Enumeration Type Documentation

◆ Flags

enum Ares::WindowSettings::Flags : uint16_t

These flags can be combined to set various properties for the window such as windowed or fullscreen mode, borderless, and whether the window is resizable or maximized. The flags use bitwise operations for easy combination.
